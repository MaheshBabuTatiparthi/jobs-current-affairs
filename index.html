Current Affairs and Job News

JWT Authentication

GET https://cajobs.herokuapp.com/restapi.php?api=jwt-auth


Request Token

POST https://cajobs.herokuapp.com/restapi.php?api=jwt-auth&action=token

Parameters

ParameterValueDescriptionData Type
usernamejasmanusername of userString
password123456password of userString

Validates the user credentials, username and password, and returns a token to use in a future request to the API if the authentication is correct or error if the authentication fails



Token Validation

POST https://cajobs.herokuapp.com/restapi.php?api=jwt-auth&action=token-validate

This is token validator; you only will need to make a POST request sending the Authorization header.



List Categories

GET https://cajobs.herokuapp.com/restapi.php?api=categories

Parameters

ParameterValueDescriptionData Type
sortasc | ascOrder sort attribute ascending or descendingString
orderbyid |
category-name |
category-image |
random		Sort collection by object attributeString
idLimit result set to items with more specific by `id`Integer
category-nameLimit result set to items with more specific by `category_name`String
category-imageLimit result set to items with more specific by `category_image`String

Example

- https://cajobs.herokuapp.com/restapi.php?api=categories
- https://cajobs.herokuapp.com/restapi.php?api=categories&orderby=id
- https://cajobs.herokuapp.com/restapi.php?api=categories&orderby=random
- https://cajobs.herokuapp.com/restapi.php?api=categories&orderby=id&sort=desc



Retrieve a Category 

GET https://cajobs.herokuapp.com/restapi.php?api=categories&id=<id>

Parameters

ParameterValueDescriptionData Type
idUnique identifier for the object and use random for random IDsInteger

Example

- https://cajobs.herokuapp.com/restapi.php?api=categories&id=1
- https://cajobs.herokuapp.com/restapi.php?api=categories&id=2
- https://cajobs.herokuapp.com/restapi.php?api=categories&id=random
